NAME
       getwc — get a wide character from a stream

SYNOPSIS
       #include <stdio.h>
       #include <wchar.h>

       wint_t getwc(FILE *stream);

DESCRIPTION
       The  functionality  described  on this reference page is aligned with the ISO C standard. Any conflict between
       the requirements described here and the ISO C standard is unintentional. This volume of POSIX.1‐2008 defers to
       the ISO C standard.

       The getwc() function shall be equivalent to fgetwc(), except that if it is implemented as a macro it may eval‐
       uate stream more than once, so the argument should never be an expression with side-effects.

RETURN VALUE
       Refer to fgetwc().

ERRORS
       Refer to fgetwc().

       The following sections are informative.

EXAMPLES
       None.

APPLICATION USAGE
       Since it may be implemented as a macro, getwc() may treat incorrectly a stream argument with side-effects.  In
       particular,  getwc(*f++) does not necessarily work as expected.  Therefore, use of this function is not recom‐
       mended; fgetwc() should be used instead.
