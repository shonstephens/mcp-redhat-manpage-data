NAME
       isatty - test whether a file descriptor refers to a terminal

SYNOPSIS
       #include <unistd.h>

       int isatty(int fd);

DESCRIPTION
       The isatty() function tests whether fd is an open file descriptor referring to a terminal.

RETURN VALUE
       isatty()  returns 1 if fd is an open file descriptor referring to a terminal; otherwise 0 is returned, and er‐
       rno is set to indicate the error.

ERRORS
       EBADF  fd is not a valid file descriptor.

       EINVAL fd refers to a file other than a terminal.  POSIX.1 specifies the error ENOTTY for this case.
