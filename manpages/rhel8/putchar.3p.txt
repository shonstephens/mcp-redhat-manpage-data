NAME
       putchar — put a byte on a stdout stream

SYNOPSIS
       #include <stdio.h>

       int putchar(int c);

DESCRIPTION
       The  functionality  described  on this reference page is aligned with the ISO C standard. Any conflict between
       the requirements described here and the ISO C standard is unintentional. This volume of POSIX.1‐2008 defers to
       the ISO C standard.

       The function call putchar(c) shall be equivalent to putc(c,stdout).

RETURN VALUE
       Refer to fputc().

ERRORS
       Refer to fputc().
