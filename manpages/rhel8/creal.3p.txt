NAME
       creal, crealf, creall — complex real functions

SYNOPSIS
       #include <complex.h>

       double creal(double complex z);
       float crealf(float complex z);
       long double creall(long double complex z);

DESCRIPTION
       The  functionality  described  on this reference page is aligned with the ISO C standard. Any conflict between
       the requirements described here and the ISO C standard is unintentional. This volume of POSIX.1‐2008 defers to
       the ISO C standard.

       These functions shall compute the real part of z.

RETURN VALUE
       These functions shall return the real part value.

ERRORS
       No errors are defined.

       The following sections are informative.

EXAMPLES
       None.

APPLICATION USAGE
       For a variable z of type complex:

           z == creal(z) + cimag(z)*I

RATIONALE
       None.

       None.
