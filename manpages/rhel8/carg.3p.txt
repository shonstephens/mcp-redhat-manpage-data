NAME
       carg, cargf, cargl — complex argument functions

SYNOPSIS
       #include <complex.h>

       double carg(double complex z);
       float cargf(float complex z);
       long double cargl(long double complex z);

DESCRIPTION
       The  functionality  described  on this reference page is aligned with the ISO C standard. Any conflict between
       the requirements described here and the ISO C standard is unintentional. This volume of POSIX.1‐2008 defers to
       the ISO C standard.

       These functions shall compute the argument (also called phase angle) of z, with a branch cut along  the  nega‐
       tive real axis.

RETURN VALUE
       These functions shall return the value of the argument in the interval [−π, +π].

ERRORS
       No errors are defined.
