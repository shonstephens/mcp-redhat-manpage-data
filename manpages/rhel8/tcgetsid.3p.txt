NAME
       tcgetsid — get the process group ID for the session leader for the controlling terminal

SYNOPSIS
       #include <termios.h>

       pid_t tcgetsid(int fildes);

DESCRIPTION
       The  tcgetsid()  function shall obtain the process group ID of the session for which the terminal specified by
       fildes is the controlling terminal.

RETURN VALUE
       Upon successful completion, tcgetsid() shall return the process group ID of the session  associated  with  the
       terminal. Otherwise, a value of −1 shall be returned and errno set to indicate the error.

ERRORS
       The tcgetsid() function shall fail if:

       EBADF  The fildes argument is not a valid file descriptor.

       ENOTTY The calling process does not have a controlling terminal, or the file is not the controlling terminal.
