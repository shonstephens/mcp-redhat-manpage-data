NAME
       catclose — close a message catalog descriptor

SYNOPSIS
       #include <nl_types.h>

       int catclose(nl_catd catd);

DESCRIPTION
       The  catclose()  function shall close the message catalog identified by catd.  If a file descriptor is used to
       implement the type nl_catd, that file descriptor shall be closed.

RETURN VALUE
       Upon successful completion, catclose() shall return 0; otherwise, −1 shall be returned, and errno set to indi‐
       cate the error.

ERRORS
       The catclose() function may fail if:

       EBADF  The catalog descriptor is not valid.

       EINTR  The catclose() function was interrupted by a signal.
